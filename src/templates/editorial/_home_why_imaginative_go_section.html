Why Imaginative Go?

Ideas about achieve something with Go lang

This project will give you some ideas about how to achieve something with Go language by looking into the code inside this imaginative web application project.

Plenty of working code samples

Actually now still small amount of sample included hahaha, but don't worry we will add it over time. Just sit back and relax, drink your coffee.

Easy to run (only with Docker)

We design it to be simple. You only need Docker to run this application. Even you don't need Go installed on your system. But you can also run with your local setup if you want.
